## Git的基本操作

###### 1.git add 添加到缓冲区

###### 2.git commit -m 提交到版本库（一次全部提交）

###### 3.git status 查看仓库当前的状态

###### 4.git diff 查看修改（比较有用）

###### 6.git log 查看日志（还有git log –pretty=oneline）

###### 7.git reset –hard HEAD^  退回到前一个版本（HEAD~100退回到前100次版本）

###### 8.git reflog 查看使用过的命令

###### 9.git checkout 撤回修改

###### 10.git rm 删除文件
